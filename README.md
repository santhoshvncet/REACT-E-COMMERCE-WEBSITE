## Run Locally

Clone the project

```bash
 https://github.com/santhoshvncet/REACT-E-COMMERCE-WEBSITE
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
